# Lotto Number Generator & AI Animal Face Test

A modern, responsive web application featuring a lottery number generator and an AI-powered animal face classifier.

## 🚀 Features

*   **Lotto Generator:** 6 unique numbers (1-45) with animated UI.
*   **AI Animal Face Test:** Classifies user photos as Dog or Cat types using Teachable Machine.
*   **Theme Support:** Light and Dark modes with LocalStorage persistence.
*   **Feedback & Partnership:** Integrated Formspree forms.
*   **Comments:** Disqus integration for community interaction.

## 🛠 Tech Stack

*   Vanilla HTML/CSS/JS
*   [Teachable Machine](https://teachablemachine.withgoogle.com/) (TensorFlow.js)
*   [Formspree](https://formspree.io/)
*   [Disqus](https://disqus.com/)

### Cloudflare Pages (Setup Guide)
To deploy this project to Cloudflare Pages:
1.  Log in to the [Cloudflare Dashboard](https://dash.cloudflare.com/).
2.  Navigate to **Workers & Pages** > **Create application** > **Pages** > **Connect to Git**.
3.  Select the `parksamkoo39-coder/samkoo_test` repository.
4.  Set the **Build settings**:
    *   **Framework preset:** None
    *   **Build command:** (Leave empty)
    *   **Build output directory:** (Leave empty or `/`)
5.  Click **Save and Deploy**.
